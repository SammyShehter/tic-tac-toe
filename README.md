# Tic Tac Toe #

Tested on Linux and Windows
***
#### To run: ####
- First be sure that docker and docker-compose are installed on your machine
- Run commands below:
```sh
cd docker-compose
docker-compose up -d
```
***
#### Verify the deployment by navigating to your server address in ####
#### your preferred browser. ####

```sh
127.0.0.1:3000
```
